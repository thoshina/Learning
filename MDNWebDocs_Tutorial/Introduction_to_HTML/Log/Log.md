#### HTML を始めよう
HTML は要素の集まりからなり、コンテンツの一部を要素で囲む（マークアップする）ことで、特定の見せ方・動かし方を表現する

##### HTML内の空白

HTMLではどれだけ半角スペースを入れても一つのスペースと認識される。
`<p>` 要素内で改行しても、半角スペースになる。改行する場合は `<br>` を使う。
